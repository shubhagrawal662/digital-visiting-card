# Fresh Basket Brand Guide

## Positioning
Premium farm-to-home grocery experience for Indian households.

## Color palette
- Primary Green: `#1B5E20`
- Accent Green: `#43A047`
- Soft Mint: `#E8F5E9`
- Deep Text: `#263238`
- Surface White: `#FFFFFF`

## Typography
- Heading: Inter / Poppins Semibold
- Body: Inter Regular

## UI style
- Rounded corners (12px-16px)
- Light shadows, airy spacing
- Micro animations for cart and status updates
- Dark mode parity for all screens
